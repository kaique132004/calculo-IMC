# Calculadora de IMC (Índice de Massa Corporal)

Este é um programa Java simples que calcula o Índice de Massa Corporal (IMC) com base na altura e no peso fornecidos pelo usuário. O IMC é uma medida comum para avaliar se alguém está em um peso saudável.

## Pré-requisitos

- Java JDK (Java Development Kit) instalado em seu sistema.

## Como Usar

1. Clone ou faça o download deste repositório.

2. Abra um terminal (prompt de comando) e navegue até o diretório onde o arquivo `aula_mauricio.java` está localizado.

3. Compile o programa usando o comando:

   ```bash
   javac aula_mauricio.java
   ```

4. Execute o programa usando o comando:

   ```bash
   java aula_mauricio
   ```

5. O programa solicitará que você insira sua altura em metros e seu peso em quilogramas.

6. O programa calculará o seu IMC e mostrará uma mensagem correspondente à sua faixa de IMC.

## Exemplo de Uso

```
Digite sua altura (em metros):
1.78
Digite seu peso (em quilogramas):
75.5
Peso Ideal: 23.81
```
